## Super Dispatch API Access

To get access to the **Super Dispatch API**, first ensure API access is included in your subscription, then request credentials from their support team.

### How to Obtain API Access

<Steps>
  <Step title='Verify Subscription'>
    The **Shipper API** is normally available for *$250/month*. When used through BeRocker, this fee
    is waived â€” giving you full API access at **no additional cost**.
  </Step>
  <Step title='Request Credentials'>
    Contact Super Dispatch technical support to receive your `ClientID` and `ClientSecret`.
  </Step>
</Steps>

### Contact Super Dispatch Support

- Email: `support@superdispatch.com`
- Phone: `+1 (816) 974-7002`
- In-App: Use the chat icon in the bottom-right corner of the Carrier TMS or Shipper TMS dashboard.

### Add Credentials in BeRocker

<Steps>
  <Step title='Open Super Dispatch Integration Settings'>
    In BeRocker, go to `Settings -> Integrations -> Super Dispatch`.
  </Step>
  <Step title='Enter API Credentials'>
    Map the fields as follows:
    - **Username**: Enter your `ClientID`
    - **Password**: Enter your `ClientSecret`
  </Step>
</Steps>

### Pricing Insights (Optional)

`X-API-KEY` is used for Super Dispatch Pricing Insights service.

If you have this subscription (or want to request it), contact Super Dispatch using the same support channels above, then enter your key in the `X-API-KEY` field.
